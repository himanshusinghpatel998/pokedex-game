# 🏆 Pokedex Game

## 🚀 Getting Started

### 🗄️ Start the Database Server

Run the following command to start **MySQL inside Docker**:

```sh
docker-compose up -d
```

### 📦 Install Dependencies and Start the Backend Server

```sh
cd backend
npm i
npm run dev

```

### 📦 Install Dependencies and Start the Frontend Server

```sh
cd frontend
npm i
npm run dev

```
